# CPE-461-462-Gobilda-Robot-ESP-Motor-Encoder

to run use VS Code with Platform IO extension (as well as downloading FreeRTOS)

1. ensure all files are green and runnable
2. click 'check' icon at bottom left of VS Code
3. then click the 'upload' icon
4. last click the 'plug' icon and the serial monitor should show the ticks

* validate the circuitry is wired correctly so board doesn't get fried
* right and left wheels match w/ repsect to robot
